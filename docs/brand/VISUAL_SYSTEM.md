# Brand Kit Studio Visual System

## Visual Concept

A premium creative operating room: calm, bright, practical, and exact. It should feel like a brand lab connected to a project command center.

## Color Palette

| Role | Name | Hex |
| --- | --- | --- |
| Background | Paper | `#f8f8f5` |
| Surface | White | `#ffffff` |
| Text | Ink | `#141412` |
| Primary | Deep Teal | `#22577a` |
| Hover/Depth | Night | `#102a37` |
| Accent | Signal Coral | `#ff6b4a` |
| Secondary Accent | Saffron | `#e8a71c` |

## Typography

- Heading: Inter Tight, Inter, system-ui, sans-serif
- Body/UI: Inter, system-ui, sans-serif
- Code/folder paths: system monospace

## UI Rules

- Use a three-panel workflow when possible: input, preview, export.
- Keep panels crisp and un-nested.
- Use real controls, not fake marketing sections.
- Keep labels small, uppercase, and sparse.
- Prefer clear borders over heavy decoration.

## Do Not Use

- Purple-blue gradient dominance.
- Beige-heavy palette.
- Decorative blobs, orbs, or generic SaaS card grids.
- Fake metrics, fake integrations, or fake proof.
