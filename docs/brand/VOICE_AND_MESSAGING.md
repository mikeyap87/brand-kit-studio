# Brand Kit Studio Voice And Messaging

## Voice Principles

- Practical first.
- Premium but not precious.
- Clear about what is local, draft, approved, exported, or still missing.
- Honest about provider and automation limits.

## Words To Use

- Brand source of truth
- Project-ready
- Export pack
- Design tokens
- Owner approval
- Local-first
- Draft
- Install into project

## Words To Avoid

- Magic
- Viral
- Guaranteed
- Fully autonomous
- Final trademark-ready

## One-Liner

Brand Kit Studio turns rough project inputs into a complete brand package Codex can keep using.

## Short Bio

Brand Kit Studio is a local-first brand kit generator for creating brand guides, visual systems, messaging rules, starter assets, design tokens, and Codex handoff prompts for new projects.
